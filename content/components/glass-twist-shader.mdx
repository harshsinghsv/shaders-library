## Installation

**Option 1: Using CLI (Recommended)**

Install this shader directly using the CLI:

```bash
npx shaderz add
```

Select "Glass Twist" from the list. The component will be added to `/components/shaderz/GlassTwistShader.tsx`.

**Option 2: Manual Installation**

Copy the component code from the Usage section below.

## Required Dependencies

```bash
npm install three @types/three framer-motion
```

## Features

- **Glass Effect**: Transparent ribbons with refraction
- **Fresnel Lighting**: Edge reflections like real glass
- **Caustic Patterns**: Light patterns through glass
- **Light Background**: Soft gray-white gradient
- **Performance Optimized**: Efficient WebGL implementation
