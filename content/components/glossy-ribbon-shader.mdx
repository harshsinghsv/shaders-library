## Installation

**Option 1: Using CLI (Recommended)**

Install this shader directly using the CLI:

```bash
npx shaderz add
```

Select "Glossy Ribbon" from the list. The component will be added to `/components/shaderz/GlossyRibbonShader.tsx`.

**Option 2: Manual Installation**

Copy the component code from the Usage section below.

## Required Dependencies

```bash
npm install three @types/three framer-motion
```

## Features

- **Twisted Ribbons**: Multiple flowing ribbon layers
- **Glossy Finish**: Specular highlights for shiny appearance
- **Rich Colors**: Magenta to purple to blue gradients
- **Edge Lighting**: Subtle edge glow effects
- **Performance Optimized**: Efficient WebGL implementation
