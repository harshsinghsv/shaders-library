## Installation

**Option 1: Using CLI (Recommended)**

Install this shader directly using the CLI:

```bash
npx shaderz add
```

Select "Neon Fluid" from the list. The component will be added to `/components/shaderz/NeonFluidShader.tsx`.

**Option 2: Manual Installation**

Copy the component code from the Usage section below.

## Required Dependencies

```bash
npm install three @types/three framer-motion
```

## Features

- **Realistic Fire**: Flowing flames with natural upward motion
- **Dynamic Colors**: Gradient from dark red to orange to yellow
- **Turbulent Motion**: Noise-based flame turbulence
- **Glowing Tips**: Bright white-yellow flame tips
- **Performance Optimized**: Efficient WebGL implementation
