## Installation

Install the required dependencies:

```bash
npm install three @types/three
```

Or with yarn:

```bash
yarn add three @types/three
```

Or with pnpm:

```bash
pnpm add three @types/three
```

## Features

- **Realistic Fire**: Flowing flames with natural upward motion
- **Dynamic Colors**: Gradient from dark red to orange to yellow
- **Turbulent Motion**: Noise-based flame turbulence
- **Glowing Tips**: Bright white-yellow flame tips
- **Performance Optimized**: Efficient WebGL implementation
